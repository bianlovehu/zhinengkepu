# 客服政策手册

# 7天无理由退换货
支持符合条件的商品在签收后7天内申请无理由退货或换货。商品需保持不影响二次销售：商品、配件、赠品、说明书、发票和原包装尽量齐全，未明显使用、未人为损坏、未拆洗或改装。定制类、食品生鲜、贴身用品、已激活软件或页面明确不支持无理由的商品，不适用7天无理由。

无理由退换货的寄回运费通常由买家承担；若是商品质量问题、错发、少发、运输破损或商家责任导致退换货，往返运费由商家承担。退款在仓库签收并验收通过后处理，通常1-3个工作日发起退款，款项按原支付方式退回；银行卡或信用卡到账时间以银行处理为准，通常3-7个工作日。

# 质量问题退换货和售后
商品存在质量问题、与描述严重不符、收到即损坏、少发、错发、疑似二手或假货时，应先安抚用户并引导提供订单号、商品照片或视频、外包装照片、物流面单和问题描述。核实属实后，支持退货退款、换货、补发或维修，责任运费由商家承担。

已超过7天无理由期限的商品，若不存在质量问题，通常不支持无理由退货；若仍在质保期内且属于非人为质量故障，可按售后政策维修或换货。轻微划痕、包装破损但不影响使用时，可以申请售后核实；是否换货取决于商品状态和责任原因。

# 运费承担
商家责任包括质量问题、错发、少发、漏发、运输破损、页面宣传与实物严重不符、承诺服务未履行，这类场景退换货、补发或维修的合理运费由商家承担。用户个人原因包括不喜欢、拍错、尺寸不合适、无质量问题退货，寄回运费通常由用户承担。涉及垫付运费时，应说明保留快递凭证，核实后按规则报销。

# 发票
商品支持开具发票。常见类型包括电子普通发票、纸质普通发票，部分企业采购可根据平台能力申请增值税专用发票。开票信息通常包括个人或公司抬头、纳税人识别号、邮箱、收票地址等。电子发票一般在订单完成或发货后1-3个工作日发送至邮箱或订单页面；纸质发票随货或单独寄出。

发票抬头、税号或内容开错时，可以申请重开。用户需提供原发票、正确开票信息和订单号；原发票作废后重新开具。若订单已退款，发票需作废或冲红后再处理。

# 物流配送
商品支持配送到大部分乡镇，具体是否可达取决于收货地址和快递覆盖范围。乡镇地区通常不额外加运费，偏远地区或大件商品可能产生附加费用，以订单页或客服核实为准。一般下单后48小时内发货，城市地区约2-3天送达，乡镇约3-5天，偏远地区可能5-7天。

物流显示待揽收通常表示商品已打包出库，等待快递员上门揽件。一般24小时内会更新；超过24小时未揽收，应为用户催促仓库或快递。快递丢失时，应先安抚用户，联系物流核查；确认丢件后支持补发或退款，通常3-7个工作日内给出处理结果。

快递派送时用户不在家，可联系快递员改约时间、放驿站或按地址再次派送。签收后发现运输破损，也可以申请售后；建议保留外包装、面单和商品损坏照片，便于向快递追责。

# 包装破损、少发、错发
收到外包装破损时，应先拍摄外包装、面单、内件和破损位置照片。如果商品完好且不影响使用，可根据用户意愿申请补偿或记录；若商品损坏或影响二次销售，支持退换货，责任运费由商家或物流承担。

少发、漏发、错发时，用户需提供开箱照片、订单明细和实收商品照片。核实后少发商品优先补寄，错发商品可安排换回正确商品。由商家责任导致的补寄和换货不应让用户承担运费。

# 二手、拆封、假货、虚假宣传
若用户反馈商品被拆封、有污渍、疑似二手，应先致歉并要求提供开箱照片、封签、商品细节和订单号。核实属实后支持退货退款或换新，并承担运费。

若用户反馈疑似假货或宣传功能与实际不符，应升级人工客服或售后专员核查。用户可提供防伪验证结果、商品照片、宣传页面截图和问题说明。核实属实后支持退货退款、换货或依法依规处理赔偿。

# 质保、维修和人为损坏
商品在质保期内出现非人为质量故障，支持免费检测和维修；如需更换配件且属于质保范围，不应收取配件费。人为损坏、进水、摔坏、私自拆修、错误使用、超过质保期等情况，可以提供付费维修，费用以检测报价为准，维修前需征得用户同意。

维修周期通常为7-15个工作日，复杂故障或需等配件时可能延长，应主动说明进度。维修后短期内出现同样故障，且确认是上次维修不彻底导致，应免费重新维修，并合理延长维修质保期。

# 投诉安抚话术
遇到投诉、辱骂、服务差、补发超时、维修超时、快递员态度差等问题，先表达歉意，再说明会记录并升级处理，最后给出明确流程和所需材料。话术建议：非常抱歉给您带来不便，这个情况我们会优先为您处理。请您提供订单号、问题照片或视频、物流单号和具体诉求，我们会核实责任；若属商家或物流责任，将为您安排退款、换货、补发、维修或投诉跟进。

回答投诉题时不要只说“正在核实”或“请稍后”，必须说明下一步、材料、责任判断和可选处理方案。
